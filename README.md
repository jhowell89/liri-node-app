<p>Use node to run this program. Use node liri.js then run one of the following comands, then add search text:</p>
<ul>
<li><code>concert-this</code></li>
<li><code>spotify-this-song</code></li>
<li><code>movie-this</code></li>
<li><code>do-what-it-says</code></li>
</ul>

<p>Example:
node liri.js movie-this underworld</p>

<ul>
<li>Artist(s)</li>
<li>The song's name</li>
<li>A preview link of the song from Spotify</li>
<li>The album that the song is from</li>
<li>If no song is provided then your program will default to "The Sign" by Ace of Base.</li>

<h3><a id="user-content-when-movie-this-command-is-used-you-will-be-provided-with" class="anchor" aria-hidden="true" href="#when-movie-this-command-is-used-you-will-be-provided-with"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>When movie-this command is used you will be provided with:</h3>
<ul>
<li>Title of the movie.</li>
<li>Year the movie came out.</li>
<li>IMDB Rating of the movie.</li>
<li>Rotten Tomatoes Rating of the movie.</li>
<li>Country where the movie was produced.</li>
<li>Language of the movie.</li>
<li>Plot of the movie.</li>
<li>Actors in the movie.</li>
<li>If the user doesn't type a movie in, the program will output data for the movie 'Mr. Nobody.</li>
</ul>
